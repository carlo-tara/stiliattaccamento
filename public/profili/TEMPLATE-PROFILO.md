# Template per i 12 Profili

Questo documento descrive la struttura standard per ogni profilo dei 12 profili di attaccamento.

## Struttura Standard di Ogni Profilo

Ogni profilo HTML segue questa struttura:

1. **Hero Section**
   - Nome profilo (es. "ANSIOSO ALTO")
   - Archetipo (es. "Il Matto (0) - L'Ingenuità Traumatizzata")
   - Arcano Maggiore + Archetipo Junghiano
   - Badge livello (Basso/Medio/Alto)
   - Colore pastello distintivo dello stile

2. **L'Archetipo Junghiano** (da archetipi-junghiani.md)
   - L'Immagine Archetipale
   - L'Archetipo Principale
   - Il Corpo Archetipale
   - L'Ombra

3. **Il Profilo nel Modello a Gradienti** (da gradienti-3livelli.md)
   - Caratteristiche del Livello (Basso/Medio/Alto)
   - Nel Corpo (sensazioni fisiche)
   - Pattern Relazionale
   - Trigger

4. **La Consapevolezza Archetipale**
   - Il Primo Passo della Consapevolezza
   - Timeline Archetipale (mese 0, 3, 6, 9, 12+)
   - L'Integrazione Archetipale

5. **Esercizi Specifici** (da gradienti-3livelli.md)
   - Lista esercizi per questo livello
   - Frequenza consigliata
   - Quando cercare terapia

6. **Navigazione**
   - Link a livello precedente/successivo dello stesso stile
   - Link a 4 Stili Base
   - Link a Archetipi
   - Link a Esercizi

## Note per Implementazione

- Ogni profilo usa il colore pastello corrispondente allo stile
- Le sezioni archetipo sono tratte da `docs/archetipi-junghiani.md`
- Le sezioni gradienti sono tratte da `docs/stili-attaccamento-gradienti-3livelli.md`
- Gli esercizi sono specifici per livello e stile
- La navigazione permette di muoversi tra i livelli dello stesso stile
